# Awake 2.0.10

Awake now installs one persistent launchd-supervised runtime. Turning Awake on or off no longer rewrites, reloads, or waits for a LaunchAgent.

- repeated backend measurements complete in about 173 ms median for on and 133 ms median for off on the test Mac
- runtime crashes preserve active leases and reconcile after launchd restarts the process
- concurrent CLI/runtime reconciliation is serialized so an off action cannot leave stale awake state behind
- timers work on the stock macOS Bash 3.2 and replace an existing indefinite manual session
- automatic agent protection is now a preference; the safety runtime remains supervised
- the native window title and zoom control are hidden so they do not overlap the close control
