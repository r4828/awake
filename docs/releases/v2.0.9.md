# Awake 2.0.9

Every active Awake lease now has a launchd-supervised battery monitor. Awake waits for the monitor's heartbeat before it keeps a session awake, and launchd restarts it after an unexpected exit.

Awake forces sleep when battery reaches the configured critical floor. The default floor is 5%.

An upgrade also checks existing leases. If Awake cannot verify a monitor, it restores normal sleep instead of leaving a session active.
