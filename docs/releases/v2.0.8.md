# Awake 2.0.8

Awake now explains how to leave blackout mode before your screens go dark.

## Blackout recovery

- The first blackout shows “Press Option + 1 anytime to show your screens” for five seconds.
- `Option+1` still toggles blackout on later uses.
- `Escape` provides a second exit while Awake owns the screen.
- `awake screens show` restarts the menu app and restores any display left at zero brightness.

## Display safety

- Awake uses its black overlay on the built-in MacBook display instead of setting that panel to zero brightness through a private API.
- Awake removes blackout windows before restoring external-monitor brightness.
- Awake restores blackout state when the menu app quits.
- Awake no longer moves your pointer or drags its menu bar icon during launch.

## Install or update

```bash
npx --yes awake-agent@latest install
```
